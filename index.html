How to Use

1. Enter the prop pitch

Use the pitch stamped on the propeller, usually shown in inches.

2. Enter gear ratio

Use the lower-unit ratio as a number, such as 1.75 for 1.75:1.

3. Add RPM and GPS speed

Use engine RPM and actual GPS speed from the same run for the best estimate.

4. Read the slip result

Lower slip often indicates a more efficient setup, but hull, load, trim, water, and prop style all matter.

What is Propeller Slip?

Propeller slip is the gap between how far the propeller should advance in theory and how far the boat actually moves through the water. This guest calculator helps compare setup changes, prop choices, and running conditions.
